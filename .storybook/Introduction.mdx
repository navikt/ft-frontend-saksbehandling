# 📖 Introduksjon til komponentbiblioteket

Dette er dokumentasjonen for komponentbiblioteket vårt, utviklet med **React**, **TypeScript**, og **Vite**. Biblioteket inneholder en rekke felleskomponenter og hjelpefunksjoner organisert i pakker som brukes av Team Foreldrepenger og Team Sykdom i Familien.

## 📦 Pakker

Biblioteket er strukturert i flere pakker for ulike formål:

### Fakta-moduler
- `@navikt/ft-fakta-beregning`
- `@navikt/ft-fakta-fordel-beregningsgrunnlag`
- `@navikt/ft-fakta-tilbakekreving-feilutbetaling`

### Prosess-moduler
- `@navikt/ft-prosess-beregningsgrunnlag`
- `@navikt/ft-prosess-tilbakekreving`
- `@navikt/ft-prosess-tilbakekreving-foreldelse`
- `@navikt/ft-prosess-tilbakekreving-vedtak`

### Skjema og validering
- `@navikt/ft-form-hooks`
- `@navikt/ft-form-validators`

### UI og plattform
- `@navikt/ft-plattform-komponenter`
- `@navikt/ft-ui-komponenter`

### Grunnleggende moduler
- `@navikt/ft-hooks`
- `@navikt/ft-types`
- `@navikt/ft-utils`

### Interne pakker
- `kodeverk`
- `storybook-utils`
- `types`
- `utils-test`


## 🚀 Kom i gang

For å bruke komponentene, installer de nødvendige pakkene i ditt prosjekt. f.eks.:

```sh
yarn install @navikt/ft-form-hooks
```

## 📚 Storybook

For å kjøre Storybook lokalt:

```sh
yarn storybook
```
